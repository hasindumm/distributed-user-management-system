# Distributed User Management System

Distributed user management system built with Go, PostgreSQL, and NATS using a microservice and event-driven architecture.

## Services

- Gateway Service (REST + WebSocket)
- User Service (NATS + PostgreSQL)

## Tech Stack

Go, PostgreSQL, NATS, Docker

## Run

docker-compose up -d

## Status

Milestone 0 - Initial setup (Git repository created)
